# Conversor de Unidades
## Autor
Diego Castro Satt

## Ramas utilizadas
- **main:** rama estable del proyecto.
- **feature-interfaz:** desarrollo y mejoras de la interfaz gráfica.
- **feature-conversiones:** implementación de conversiones y lógica del sistema.
- **test:** pruebas y validaciones antes de integrar cambios a la
rama main
